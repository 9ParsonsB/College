Adding Items to Buy

To add items to your oder; simply click on the picture of the item you want to add. The items will then appear in the list on the right - including its price.


Removing items

To remove an item, simply click on the item on the list that you want to remove, then click the "Remove" button.


Creating a new order / Clearing an order

To Create a new order or clear an existing one, click the "clear" button, above the "buy" and "remove" button. Or, alternativly, click File, New.


Saving an Order

After you create an order and you would like to save your order to file (.txt), click File -> Save. A dialogue box will appear asking you where you would like to save your order



Loading a Saved Order

To load an order from file, click File -> Load and then locate and select the file you would like to load, then click "Open"



Exiting the program / Logging Out

To Exit the Program or to logout, click the X in the top left corner, or click File -> Logout. Both will take you back to the Login form.
